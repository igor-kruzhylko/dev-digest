# server/ — `@devdigest/api`

Fastify API + Drizzle/Postgres (pgvector). Repo context indexer (`repo-intel`) lives here.
Overview → [README.md](README.md).

## Commands
`pnpm dev` (:3001) · `build` · `typecheck` · `test` · `db:generate` · `db:migrate` · `db:seed`
- Test split by filename: `*.it.test.ts` = DB-backed (testcontainers Postgres); everything else hermetic.

## Conventions (non-default)
- Feature module = Fastify plugin with the triple `routes.ts → service.ts → repository.ts`
  (see `src/modules/repos` as the reference). Registered statically in `src/modules/index.ts`.
- DI via `src/platform/container.ts`: services depend on interfaces from `@devdigest/shared`,
  not concrete adapters → tests inject mocks through `overrides`.
- External world wrapped in `src/adapters/*` (github, git, llm, embedder, codeindex, …).
- Request validation + response serialization use Zod directly (`fastify-type-provider-zod`).

## Gotchas
- `repo-intel` is in-process (not a separate service) and special — its own map: [src/modules/repo-intel/CLAUDE.md](src/modules/repo-intel/CLAUDE.md).

## Do-not-touch
- `src/db/migrations/**` — generated by `pnpm db:generate`; never hand-edit.
- `src/vendor/**` — vendored contracts; see root CLAUDE.md.
- `clones/**` — third-party repos under review.
